# sreagent 登录页品牌重塑与视觉优化

**日期**：2026-06-28  
**范围**：`frontend/src/views/LoginView.vue`  
**目标**：将登录页从"lc-agent"更名为"sreagent"，全面升级视觉设计以匹配 AI+SRE 产品定位

---

## 设计概念：「智能控制面」

面向运维工程师的 AI 驱动 SRE 工具。登录页应是进入控制中心的入口，传递**精确、可靠、掌控感**，同时融入 AI 的智能质感。

### 配色方案

| 角色 | 色值 | 用途 |
|------|------|------|
| 背景 | `#090d1a` 极深靛蓝 | 主背景，接近漆黑但保有蓝色基因 |
| 面板 | `#111827` 半透明 | `rgba(17,24,39,0.85)` + backdrop-filter |
| 主强调 | `#818cf8` 靛紫 | AI 品牌色，按钮、焦点环 |
| 次强调 | `#22d3ee` 青 | 输入焦点过渡、光点闪色 |
| 告警色 | `#f59e0b` 琥珀 | 错误提示专用 |
| 正文 | `#f1f5f9` | 主要文字 |
| 辅助 | `#64748b` | 次要说明、placeholder |
| 边框 | `rgba(129,140,248,0.2)` | 卡片边框 |
| 输入底 | `#0f172a` | 输入框背景 |

### 排版

| 角色 | 字体 | 说明 |
|------|------|------|
| 品牌名 / 标题 | JetBrains Mono, monospace | 等宽字体呼应终端/CLI，字重 600 |
| UI 文字 | system-ui, -apple-system, sans-serif | 系统默认，保证可读性 |

### 品牌名显示

- 位置：卡片顶部，左对齐
- 形式：`→ sreagent`，箭头符号 + 等宽字体，像终端 prompt
- 字号：20px，字重 600，颜色 `#f1f5f9`
- 副标题："智能运维控制中心"，字号 13px，颜色 `#64748b`

## 签名元素：Canvas 动态背景

### 底层：终端网格
- 细线 0.5px，颜色 `#1e293b`
- 间距 40px，全视口覆盖
- 边缘有径向渐变遮蔽（CSS mask），中央区域更清晰

### 上层：AI 神经元光点
- 30-50 个光点，直径 2-4px
- 颜色范围：靛紫 `#818cf8` 到青 `#22d3ee`
- 独立慢速运动（随机方向 + 正弦波偏差）
- 距离 < 120px 时概率连线，透明度随距离衰减
- 在左侧/底部密度稍高，右上稀疏
- 鼠标 100px 内光点轻微排斥，形成交互反馈
- 帧率限制 30fps
- `visibilitychange` 暂停动画
- `prefers-reduced-motion` 时静止光点，仅保留网格

## 表单与交互

### 输入框
- 背景 `#0f172a`，圆角 `8px`
- 默认边框 `1px solid rgba(100,116,139,0.3)`，焦点过渡到 `#818cf8`
- Label 在上方：字号 13px，颜色 `#94a3b8`，字重 500
- Placeholder：字号 14px，颜色 `#475569`
- 禁用态：opacity 0.5
- 过渡：200ms ease

### 登录按钮
- 满宽，圆角 8px，高度 42px
- 默认：`#818cf8` 背景，白色文字，字重 600
- Hover：`#6366f1` + `box-shadow: 0 0 20px rgba(129,140,248,0.3)`
- Active：`scale(0.985)`
- Loading：文字 → "验证中..."
- 过渡：200ms ease

### 错误提示
- 琥珀色左边框（3px `#f59e0b`）+ `rgba(245,158,11,0.1)` 底色
- 文字："账号或密码错误，请重试"
- 出现动画：从顶部滑入，200ms

### 可访问性
- 可见焦点环：`0 0 0 2px rgba(129,140,248,0.5)` + outline-offset 2px
- Tab 顺序：用户名 → 密码 → 登录按钮
- Enter 在任一输入框可提交
- `prefers-reduced-motion` 禁用所有动画

## 测试策略

1. 人工检查：打开 `/login` 路由，确认品牌名显示为 `→ sreagent`，副标题为"智能运维控制中心"
2. 输入测试：输入空用户名/密码，确认错误提示样式正确
3. 提交测试：输入正确凭据，确认登录流程正常
4. 响应式：在 375px 宽度下确认卡片全宽，无溢出
5. Canvas：确认背景动画运行流畅，切换标签页后动画暂停
6. 减少动画：系统开启 `prefers-reduced-motion`，确认动效静止
7. 键盘导航：仅使用 Tab/Enter 完成登录流程
